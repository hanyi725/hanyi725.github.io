5.图片

基础语法：

![Alt text](图片链接 “optional title”)

alt text:图片描述 可写可不写

插入本地图片：

先在source/_posts 下新建一个asset 文件夹 然后把图片存进去

插入语法：

![alt text](./assets/图片名字.png)

在括号里写图片相对路径就完事 但是不好分享不好更改 如果不小心图片不在那里了 你的显示就GG了

插入网络图片：

在括号里也是写图片链接 但是比较依赖网络 人家要是没了 显示自然就没了

把图片存入markdown文件：（这个比较好 需要用Python）

就是把图片转成base64再转回来，

按道理来说 要写到辣个括号里面 但是吧从一个颜控（划掉 改为实用）的角度来看 很丑的 一般情况下用一个id代替一下下，会看起来舒服很多，

9.列表

无序列表

无序列表用 - + * 任何一种都可以
举个栗子：

\+ 你是小可爱

\- 我是小可爱

\* 我们在一起吧

注意：符号和内容直接要有空格

输出：

  • 你是小可爱
  • 我是小可爱
  • 我们在一起吧

    有序列表

    数字加点加空格加内容

1.你是小可爱

2.我是小可爱

3.不 你不是

输出：

  1. 你是小可爱

  2. 我是小可爱

  3. 不 你不是

然后多级列表比上一级多敲三空格就OK

  1. 你是谁
    1. 我是谁
    2. 不知道
  2. 你是谁
    1. 不告诉你
    2. 你猜啊

一定要记得空格！！！
