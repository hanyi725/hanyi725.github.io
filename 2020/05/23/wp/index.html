3.backup

X老师忘记删除备份文件，他派小宁同学去把备份文件找出来,一起来帮小宁同学吧！
这道题说明要找备份文件 然后联系题目backup
备份文件后缀是.bak
打开容器发现：你知道index.php的备份文件名吗？
于是输入 /index.php.bak

4.cookie

说明要查看cookie

于是/cookie.php 发现提示 See the http response，然后查看响应 可以在F12->网络里面查 也可以用bp抓包 截断 行动 发给重发器 查看响应 然后得到flag：

cyberpeace{75a6a815cb658e3e457a941bf490c1c3}

5.disabled_button

点开是一个叫flag 的按钮 但是正如题所说 不能摁 于是F12 改属性
input disabled=”” class=”btn btn-default” style=”height:50px;width:200px;” type=”submit” value=”flag” name=”auth”
把disabled 改成enabled 或者删掉 然后按钮就可以按了 得到flag：

一个位置 用狙击手 选择常用弱密码 然后爆破（点长度可排序）长度不一样的查看响应 发现flag：cyberpeace{85ec0a5938813189aa8f6de33cc180a4}

11.command_execution

打开是一个ping 然后随便输入127.0.0.1 哦 可以输入指令 只有一行 于是考虑命令拼接，（Linux指令）然后输入：127.0.0.1 && find / -name flag
看到flag.txt 在home文件夹下 然后用cat 指令 ：127.0.0.1 &&cat /home/flag.txt
得到flag ：cyberpeace{d289d0e6f13c916fc7b4906194f839a7}
