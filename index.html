5.图片

基础语法：

插入本地图片：

先在source/_posts 下新建一个asset 文件夹 然后把图片存进去

插入语法：

![alt text](./assets/图片名字.png)

在括号里写图片相对路径就完事 但是不好分享不好更改 如果不小心图片不在那里了 你的显示就GG了

插入网络图片：

在括号里也是写图片链接 但是比较依赖网络 人家要是没了 显示自然就没了

9.列表

无序列表

无序列表用 - + * 任何一种都可以
举个栗子：

\+ 你是小可爱

\- 我是小可爱

\* 我们在一起吧

注意：符号和内容直接要有空格

输出：

  • 你是小可爱
  • 我是小可爱
  • 我们在一起吧

    有序列表

    数字加点加空格加内容

1.你是小可爱

2.我是小可爱

3.不 你不是

输出：

  1. 你是小可爱

  2. 我是小可爱

  3. 不 你不是

然后多级列表比上一级多敲三空格就OK

  1. 你是谁
    1. 我是谁
    2. 不知道
  2. 你是谁
    1. 不告诉你
    2. 你猜啊

一定要记得空格！！！

First

接下来 重头戏要来了！好可怕 快跑

创建hexo

先cd 到那个有着你喜欢的名字的文件夹
bash的命令是：

cd /xx/x/x/x/

举个栗子：cd /d/github/blog/

然后输入安装hexo 命令：

npm i -g hexo

等待安装好以后 输入 hexo -v 查看hexo版本

之后输入：hexo init 初始化 hexo

这个完成后 打开辣个你喜欢的文件夹(其实就是你的博客文件夹) 就可以看到里面多了一堆东西 就说明你成功啦

third

fourth

配置 github

输入： git config --global user.name "yourname"

回车

再输入：git config --global user.email "youremail"

回车

yourxxxx就是字面意思 就是你的 别直接复制粘贴嗷

生成ssh

输入： ssh-keygen -t rsa -C "youremail"
(需要连续按三次回车)

然后我们就去寻找生成的这个鬼玩意

输入： cd ~/.ssh

然后输入：cat id_rsa.pub 就会看到你的密钥

然后点开你已经登录的github 点击你的头像旁边的辣个倒三角

然后点击setting（设置） 然后选择 SSH and GPG keys

点击 new SSH key title 随便填 然后把那个刚刚找到的密钥 粘贴过来 然后点击 add ssh key 就OK啦

然后继续回到Bash 输入：ssh -git@github.com 如果不报错 则说明成功
